### Manifesto Ágil
Fundamentos:
*  4 Valores:
  - 1 - Indivídus e Interações (mais que processos e ferramentas) / 2 - Software em funcionamento (mais que documentação abrangente) / 3 - Responder a mudanças (mais que seguir o plano) / 4 - Colaboração com o cliente (mais que negociação de contratos).
